# kmu_team3.14
프로젝트 소개

D-50은 C++로 제작된 콘솔 기반 경제 시뮬레이션 게임입니다.

플레이어는 1억 원의 빚을 진 상태에서 게임을 시작하며, 제한된 기간 안에 노동, 자격증 취득, 전문성 강화, 증강 시스템 등을 활용하여 빚을 모두 상환해야 합니다.

단순히 돈을 버는 것뿐만 아니라 이자 관리, 투자적인 선택, 성장 시스템을 통해 효율적으로 부채를 청산하는 것이 목표입니다.

게임 목표

- 1억 원의 원금을 모두 상환한다.
- 매일 발생하는 이자를 관리한다.
- 노동과 성장 시스템을 활용해 수익을 극대화한다.
- 제한된 기간 내에 빚을 청산하여 게임을 클리어한다.

| Language | C++ |
| IDE | Visual Studio 2022 |
| Platform | Windows x64 |
| Toolset | MSVC v143 |

주요 시스템

1. 채무 시스템 (Debt System)

- 원금 및 이자 관리
- 일 단위 이자 계산
- 남은 상환 기간 관리
- 상환 시 이자 우선 차감

2. 노동 시스템

타이핑 미니게임을 통해 수익을 획득할 수 있습니다.

- 노동 성공 시 보상 지급
- 하루 노동 횟수 제한
- 노동력 강화 시 보상 증가

3. 노동력 강화

장비를 구매하여 노동 효율을 높일 수 있습니다.

예시:

- 싸구려 멤브레인 키보드
- 최고급 기계식 키보드
- 인체공학 워크스테이션

강화 단계가 올라갈수록 노동 보상이 증가합니다.

4. 전문성 강화

알바 계약 및 전문성 향상을 통해 안정적인 수익을 증가시킬 수 있습니다.

5. 자격증 시스템

자격증 취득 시 기본급이 증가하여 매일 획득하는 수입이 상승합니다.

6. 증강 시스템

특정 조건 달성 시 특수 능력을 획득할 수 있습니다.

종류:

- Silver
- Gold
- Platinum

효과:

- 즉시 자금 획득
- 수익 배율 증가
- 복합 성장 효과

7. 운(Luck) 시스템

게임 진행 중 운 수치가 상승하며 더 좋은 증강 선택지가 등장할 확률이 증가합니다.
